山海经原典地图研究台 v004 / 0.4.3 覆盖更新包

使用方式：
1. 关闭正在运行的地图程序和 PowerShell 开发服务。
2. 把本压缩包全部内容解压到 F:\SHmap\SHmap。
3. Windows 提示同名文件时，选择“替换目标中的文件”。
4. 打开 F:\SHmap\SHmap\desktop-app，在 PowerShell 中运行：
   npm run verify
   npm run desktop:dev
5. 校验应显示：程序v004 / 0.4.3。

本更新包不会包含 node_modules、target、dist，也不会删除本地工作数据。
