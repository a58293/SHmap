山海经原典地图研究台 v0.5.2
世界／区域／地点层级覆盖更新说明

一、更新内容
1. 左侧索引新增“对象／世界·区域”切换。
2. 建立固定世界根节点“山海经世界”。
3. 增加世界大区与地图区域的树状目录，支持折叠、检索与定位。
4. 点击世界或区域后，地图自动缩放到完整范围，并显示区域概述、上级区域、涉及经篇和成员地点。
5. 地点详情顶部增加“世界 > 世界大区 > 地图区域 > 地块 > 地点”层级路径。
6. 低倍缩放优先显示区域名称，高倍缩放恢复具体地点，减少总览状态下的标签拥挤。
7. 区域聚焦时保留本区域对象并降低其他区域干扰。
8. 每个地点支持一个主要区域和多个关联区域，河流等跨区对象不会被强制归入单一区域。
9. 气候与生态字段已在区域数据中预留，但本版本不填写、不显示空白模块。
10. 未归入区域的对象会进入待归区结构，不会丢失或隐藏。

三、覆盖方式
将覆盖更新包解压到项目根目录 F:\SHmap\SHmap，选择覆盖同名文件。

四、运行校验
cd F:\SHmap\SHmap\desktop-app
npm run verify
npm run desktop:dev

若 node_modules 不存在，先执行：
npm ci

五、说明
地图区域是内部空间整理层，不替代《大荒东经》《海内经》等经篇分类。当前区域主要依据既有“所属区域／山系”文本及坐标方位生成，后续可以逐步改名、合并或拆分。
