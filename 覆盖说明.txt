山海经原典地图研究台 v0.4.1 更新器修复包

1. 关闭正在运行的桌面测试程序。
2. 将本压缩包内全部内容复制到仓库根目录 F:\SHmap\SHmap。
3. 选择“替换目标中的文件”。
4. 在 desktop-app 目录运行：
   npm run verify
   npm run desktop:dev
5. 测试无误后，使用 GitHub Desktop 提交并 Push origin。
6. 在 GitHub Actions 手动运行 Publish Windows Desktop Update。

注意：v0.4.0 本身的在线检查存在网络链路问题，因此首次升级到 v0.4.1 仍建议下载 v0.4.1 安装包覆盖安装。进入 v0.4.1 后，后续版本会使用多线路检查与自动重试。
