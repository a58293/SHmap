# SHmap｜山海经原典地图研究台

基于《山海经》原文、古注、平行古籍和项目推导建立的可持续扩充地图研究工具。

## 当前版本

- 程序：`Demo v013`
- 地图数据：`v031-r0001`
- 研究方法：`v002`
- 制图执行规则：`v004`

## 直接运行

进入 [`app`](app/) 文件夹，双击 `启动山海经地图Demo.bat` 或打开 `index.html`。

也可以直接打开根目录的 `index.html`，它会跳转到程序入口。

## 当前地图规则

- 都广之野中心固定为 `(0, 0)`；
- 底层坐标单位为“里”；
- 1个主格 = 100里；
- 格内按10里精度表达，底层可保留更精确坐标；
- 同一地块和同一精确坐标均允许多个对象共存；
- 普通河流使用线型表达，海、泽、湖、渊、池等水体才使用面积；
- 面积、作用域、点对象和文本归属必须分层判断。
