# 新版简述卡牌与画笔采集说明

## 宽幅地块简述

低于200%时单击有内容的地块，直接打开非全屏宽幅简述卡。左侧显示地块身份、空间关系与可信度；右侧按六类对象展示。空字段自动隐藏。

六类为：山水地貌、草木、鸟兽、金玉矿物、人群神祇、事件遗迹。分类标题仅在有对象时显示数量。

## 同坐标对象

多个对象精确坐标相同时，正面完整显示“对象甲 / 对象乙 / 对象丙”，不再用数字代替。点击右上角翻转按钮后，可以逐个切换查看与本地关系、核心特征、功效、关系说明和证据等级。

## 画笔采集

1. 点击地图工具栏“画笔采集”。
2. 按住左键拖过有对象或已有地块档案的地块。
3. 松开后自动打开简述集合。
4. “继续追加”回到地图继续采集；“擦除地块”划过已收录地块即可移除。
5. 同一地块只收录一次；高倍率下仍归并到100里主格。
6. 画笔开启时按住空格拖动，可临时移动地图。


## v027 画笔集合调整

画笔采集后不再按地块纵向排列多张卡片，而是把所有已采集地块合并为一页区域博物志。六类对象、原文、证据链和待核对问题统一汇总，底部保留采集地块目录，支持逐个定位、打开完整博物志和移除。
