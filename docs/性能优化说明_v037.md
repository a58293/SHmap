# v037 性能优化说明

本版不增加地图研究功能，也不修改对象资料、坐标或分类。

## 核心优化

1. 对象索引：对象ID、100里主格、精确锚点、面积、线型对象统一缓存。
2. 地图拖动：拖动期间复用现有DOM，仅平移地块层并刷新Canvas；松手后完整重绘一次。
3. 滚轮缩放：连续滚轮期间使用CSS变换和轻量Canvas帧，停止90毫秒后完整重绘。
4. 小地图：对象底图只在数据或尺寸变化后重画，地图移动只更新视野框。
5. 搜索与区域：搜索上下文、空间聚焦和低倍率区域聚合按条件缓存。
6. 孤立审计：只在对象数据变化后重算，不再随每一帧地图刷新。
7. 画笔：笔迹使用独立Canvas动画帧，地块采集只刷新轻量图层；完成笔画后自动抽稀。
8. 本地保存：连续操作合并为一次写入，页面隐藏或关闭时强制保存。
9. 标签避让：高倍率标签在停止移动后140毫秒执行，避免每帧读取布局。

## 兼容性

- 保留v036红色画笔、地图总览跳转、高倍率悬停关闭等功能。
- 保留本地存储键与旧工作区数据。
- 地图数据仍为v031-r0001，默认188个对象；已在浏览器中导入的扩充数据继续保留。
