
## v029 快速使用

1. 点击“画笔采集”，按住左键划过需要汇总的地块。
2. 松开后会出现与普通地块简述相同的宽幅博物志弹层。
3. 地图中的橙色已选地块可用右键直接取消。
4. 集合弹层内可继续追加、清空、逐地块移除或打开完整博物志。

## 推荐打开方式

进入 `app` 文件夹，双击：

- `启动山海经地图Demo.bat`；或
- `index.html`。

也可以直接打开：

- `山海经原典地图研究台_Demo_v018_点击灰暗区域退出聚焦_单文件版.html`。

## 当前功能

- 无限拖动的100里主格地图；
- 最高3000%缩放；
- 同格多对象与地块翻转；
- 10里格内下钻；
- 地名与人物、神明、异兽搜索；
- 搜索聚焦和面积／作用域聚焦；
- 面积与作用域可视化编辑；
- 地块博物志与十八经篇事件归档；
- Markdown格式识别、预览与导入；
- 删除、回收站、更改记录和 `.shjpatch` 导出。

## 重要说明

这是本地交互原型。浏览器会把本地修改保存在当前浏览器的本地存储中。清理浏览器数据、换浏览器或换电脑前，应先导出更改包或完整备份。

## GitHub数据与更改包

1. 使用 GitHub Desktop 把 `.shjpatch` 放入并上传：
   `submissions/pending/`
2. 打开地图，点击“检查数据更新”。
3. 在“待处理更改包”中点击“查看并应用”。
4. 程序会显示将应用、已包含和冲突的项目。
5. 只有冲突为0时，才可以写入本地地图。

同一台电脑导出后再上传的包，地图通常已经包含最终变化。此时会显示“本机已经包含”，只需标记同步，不会重复修改。

公开仓库读取无需登录；上传仍使用 GitHub Desktop。


### v019 交互补充
- 地图缩放低于200%时，单击已有内容的地块会直接进入地块博物志。
- 在地块博物志中点击中央档案页外的周边背景，或按 Esc，可返回地图。


### v021 简述卡牌

- 打开地块博物志时默认显示居中小卡牌。
- 点击右上角“完整”可展开为全屏博物志。
- 点击卡牌外部背景或按 Esc 返回地图。


## Markdown 更新地块档案

使用标题 `### 地块档案：X+002_Y+014` 指定100里主格。可录入“简要、基础、详细”等字段。导入前会预览目标主格和具体更新项；只有 Markdown 中出现的字段会改变。


### 研究工具

- 悬停地块或对象：查看透镜摘要。
- 关系高亮：以当前对象为中心显示关联对象。
- 视图预设：切换全部、地理、水系、神话、文明和证据视图。
- 对比：点击2—4个地块后并排对比。
- 测量：在地图上拖动查看距离和方向。
- 孤立对象：自动列出缺少关系信息的对象。
- 画笔集合：自动生成区域摘要。
