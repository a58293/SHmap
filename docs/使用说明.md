# 本地使用说明

## 推荐打开方式

进入 `app` 文件夹，双击：

- `启动山海经地图Demo.bat`；或
- `index.html`。

也可以直接打开：

- `山海经原典地图研究台_Demo_v013_GitHub读取接入_单文件版.html`。

## 当前功能

- 无限拖动的100里主格地图；
- 最高3000%缩放；
- 同格多对象与地块翻转；
- 10里格内下钻；
- 地名与人物、神明、异兽搜索；
- 搜索聚焦和面积／作用域聚焦；
- 面积与作用域可视化编辑；
- 地块博物志与十八经篇事件归档；
- Markdown格式识别、预览与导入；
- 删除、回收站、更改记录和 `.shjpatch` 导出。

## 重要说明

这是本地交互原型。浏览器会把本地修改保存在当前浏览器的本地存储中。清理浏览器数据、换浏览器或换电脑前，应先导出更改包或完整备份。

## GitHub版本检查

点击“检查数据更新”会读取公开仓库 `a58293/SHmap` 中的 `data/current.json`。下载读取无需登录；上传更改包仍使用 GitHub Desktop。
