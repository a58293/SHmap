# 六类内容分类审计报告 v032

## 审计范围

- 地图对象：188 个。
- 检查界面：地块简述、画笔区域博物志、悬停透镜、对象透镜、视图预设。
- 数据版本保持 `v031-r0001`，本次只修正程序的展示分类，不改研究资料和坐标。

## 新规则

1. 优先读取对象“类型”的第一层，例如 `山／神祇居所` 以“山”为主类。
2. 对象名称只在类型缺失或明确标为“未分类”时辅助判断。
3. 不再用“丹、玉、蛇、鸟、民、神、木”等单字扫描完整名称。
4. 地块级栏目可以显示对象真实字段中的资源信息；例如山体记录“白玉”，矿物栏可以显示白玉，但该山体对象仍属于地貌。
5. 旧版若自动保存过单一误分类对象名称，显示时会过滤；不会删除用户主动录入的其他文字。

## 审计结果

- 全部 188 个对象均获得且只获得一个主分类。
- 与旧规则相比，重新归类 44 个对象。
- 地貌：122 个。
- 草木：5 个。
- 鸟兽：13 个。
- 矿物独立对象：0 个。
- 人群神祇：32 个。
- 事迹：16 个。
- 另有 13 个对象在自身“矿物”字段中记有资源；这些资源可以出现在矿物栏目，但不会把山、水等对象本身改成矿物对象。

## 重点修复案例

- `帝丹朱葬`：旧版 事迹 → v032 事迹；正式类型为 `墓葬`。
- `玉山（西王母居所）`：旧版 矿物 → v032 地貌；正式类型为 `山／神祇居所`。
- `西王母（玉山居所层）`：旧版 矿物 → v032 人群神祇；正式类型为 `神祇／居所文本层`。
- `蛇山（卷十八北海层）`：旧版 鸟兽 → v032 地貌；正式类型为 `山／蛇水源山`。
- `神民之丘`：旧版 人群神祇 → v032 地貌；正式类型为 `丘`。
- `钉灵国`：旧版 鸟兽 → v032 人群神祇；正式类型为 `国家／马蹄善走之民`。
- `开明兽（昆仑九门守）`：旧版 事迹 → v032 鸟兽；正式类型为 `神兽／门守`。
- `若水（若木源段）`：旧版 草木 → v032 地貌；正式类型为 `河流源点`。

## 一致性检查

- 地块详情和悬停透镜使用同一 `objectCategory` 分类函数。
- 自动地块摘要不再通过对象名称中的单字推导矿物、草木、鸟兽或国家。
- 神话视图与文明视图已改用当前分类键，不再引用旧版 `plant / beast / deity / event` 键名。
- 空分类在悬停透镜中继续隐藏。
- 独立对象分类与对象属性分开：对象可属于地貌，同时携带矿物、草木或生灵资料。
