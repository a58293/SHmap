# Markdown 地块档案格式 v1.1

## 基本标题

```md
### 地块档案：X+002_Y+014
```

坐标是100里主格编号，必须为整数。也可在区块内另写：

```md
* 地块坐标：X+002_Y+014
```

## 支持字段

- 简要、基础、详细
- 地理环境、建筑群、生活物种
- 所属国度、信仰对象、统治者、守护神
- 奇珍异兽／栖息生物、神木／神话植被、仙草药草、丰富矿产、特殊生命、当地风俗
- 文明与神话奇遇
- 已发生事件、时间流逝、其他剧情说明
- 玩家是否可以抵达、玩家遭遇的敌人、玩家触发的剧情、玩家得到的东西
- 十八经篇事件，例如“海内经事件”“大荒北经事件”

## 多行文本

字段后的续行至少缩进两个空格：

```md
* 详细：
  第一段内容。
  第二段内容。
```

## 更新规则

- Markdown 中出现的字段会覆盖该地块对应字段。
- 明确写出但留空的字段会被清空。
- 没有写出的字段保持原值。
- 导入前会显示目标主格和具体变化。
