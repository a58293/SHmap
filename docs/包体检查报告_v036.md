# v036 包体检查报告

## 功能核对

- 画笔采集开启后，拖动路径显示红色圆角笔迹。
- 笔迹使用地图世界坐标保存，缩放和平移后仍保持原位置。
- 清空画笔集合时同步清除笔迹。
- 右上角小地图的指针事件不再被主地图拖动截获。
- 小地图单击可跳转；按住拖动可连续导航。
- 点击“地图总览”可缩放到全部已录入对象范围。
- 双击小地图也可打开全部范围总览。

- 小地图测试：点击后地图中心由原点移动到目标位置。
