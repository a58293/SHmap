# 用 GitHub Desktop 上传本首包

1. 在 GitHub Desktop 中打开已经克隆的 `SHmap` 本地仓库文件夹。
2. 将本压缩包内的全部文件复制到该仓库根目录。
3. Windows 提示同名文件时选择替换；不要复制外层压缩包文件夹本身。
4. 回到 GitHub Desktop，左侧会显示新增和修改的文件。
5. 左下角 `Summary` 填写：

   `建立山海经地图研究台仓库首包 v012`

6. 点击 `Commit to main`。
7. 点击顶部 `Push origin`。
8. 回到 GitHub 网页刷新，确认能看到 `app`、`data`、`specs`、`docs` 等目录。

不要向仓库提交密码、验证码、访问令牌、`.env`、密钥文件或个人隐私资料。
