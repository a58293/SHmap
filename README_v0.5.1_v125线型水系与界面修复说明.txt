SHmap v0.5.1 覆盖更新说明

基线：SHmap v0.5.0 特大字号与非阻挡通知修复版
目标：SHmap v0.5.1 v125数据同步与线型水系版

更新内容：
1. 对象数量由395更新为617，数据版本更新为v125-r0001。
2. 从v125坐标棋盘图118个箭头格生成79段线型水系。
3. 普通河流取消整格蓝色表达，改为连续线路、流向箭头、源点和汇入端。
4. 海、泽、湖、渊、池等面积水体继续保留浅蓝面积。
5. 悬停河流显示名称、经篇、区域、流向和G1—G4证据等级。
6. 单击河流可高亮完整路径、缩放定位，并在右侧查看水系档案。
7. 新增 npm run import:master -- "新版母表.xlsx" 母表导入命令。
8. 保留“特大”字号明显放大、工具栏增高、底部中央提示、点击穿透、2.4/4.8秒时长和最多3条提示。

覆盖方法：
把覆盖更新包解压到 F:\SHmap\SHmap，选择替换同名文件。

运行：
cd F:\SHmap\SHmap\desktop-app
npm run verify
npm run desktop:dev

预期校验：
校验：617个对象，程序v005 / 0.5.1，数据版本v125-r0001
水系路径校验：通过（79段，118个箭头格，孤立箭头0）
