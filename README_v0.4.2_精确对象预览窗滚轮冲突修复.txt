山海经原典地图研究台 v0.4.2
精确对象预览窗滚轮冲突修复

本补丁保持版本号 v004 / 0.4.2 不变。

修复内容：
1. 鼠标位于精细地图的对象预览小窗上时，滚轮只滚动小窗内部内容。
2. 小窗滚动到顶部或底部后，滚轮仍不会继续冒泡并缩放地图。
3. 鼠标移出预览小窗后，滚轮恢复为地图缩放。
4. 不改变对象点击、双击、悬停、地图缩放和自动同步 GitHub 的其他逻辑。

覆盖位置：
将压缩包内全部内容复制到 F:\SHmap\SHmap，选择替换目标文件。

测试命令：
cd F:\SHmap\SHmap\desktop-app
npm run verify
npm run desktop:dev
