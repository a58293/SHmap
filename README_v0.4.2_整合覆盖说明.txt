山海经原典地图研究台 v0.4.2 整合覆盖说明

本包把原 v0.4.3 的“无回弹缩放”和“精细对象悬停小窗”直接合并进 v0.4.2，版本号仍为 0.4.2。

覆盖位置：
F:\SHmap\SHmap

覆盖后运行：
cd F:\SHmap\SHmap\desktop-app
npm run verify
npm run desktop:dev

重点检查：
1. 地块博物志分类内容过多时，分类内部出现滚动条。
2. 连续滚轮缩放停止后不发生回弹。
3. 放大到精细地图后，悬停对象点分别出现对象预览小窗。
4. 单击小窗打开精确点博物志，双击小窗进入编辑。
5. 完成本轮编辑后自动保存并推送 GitHub。

注意：若 GitHub 上已经发布过同版本号 0.4.2，不应覆盖旧 Release；应改用更高版本号重新发布。
