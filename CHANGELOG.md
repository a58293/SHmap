# SHmap 更新记录

## GitHub待处理包下载与应用

- “检查数据更新”同时读取正式数据版本与 `submissions/pending/`。
- 新增 `.shjpatch` 在线下载、格式校验、逐项预览和本地应用。
- 支持对象新增、修改、删除、恢复，面积／作用域修改，地块档案和清空地块。
- 按更改字段检查本地冲突；存在冲突时整包不写入。
- 已应用包按 GitHub SHA 记录，避免重复应用。
- 从 GitHub 应用的内容不会再次进入本轮导出包。
- 更改记录新增“GitHub已应用包”分区。
- 修复 `.gitignore` 导致 `submissions` 中 `.shjpatch` 不显示的问题。

## 保留功能

- v013 GitHub正式数据版本检查。
- 无限地块、搜索、空间范围编辑、地块博物志和 Markdown 导入。


## v016
- 中倍率与较大地块视图改为“上方图标 + 下方当前地块名字”。
- 紧凑与微型地块仍保持纯图标，避免地图过度拥挤。


## v017
- 区域聚焦状态下，点击灰暗地块会立即退出聚焦，并切换到被点击地块。
- 点击地图空白区域会退出聚焦、清除选中状态并恢复完整地图。
- 点击聚焦区域内部仍维持当前区域聚焦。


## v018
- 经篇筛选只显示《山海经》十八经篇；组合经篇对象可在对应单篇中被筛出。
- 类型筛选改为“山、水系、国家／区域、神木／植物、神祇／人物、异兽／生物、墓葬、事件地标、其他”的折叠树。
- 地块档案新增“简要、基础、详细”三层简述与“文明与神话奇遇”。
- 地图缩放低于200%时，点击已有地块直接进入地块博物志。
- 地块博物志右上角新增“简述／完整”切换；默认简述，空白普通字段不显示。


## v019
- 修复指针事件覆盖点击事件，导致低于200%时仍翻转地块而未打开博物志的问题。
- 地块博物志中点击中央档案页以外的周边背景可返回地图。
- Esc 键可关闭地块博物志。


## v020
- 地块博物志的“简述”模式改为居中的小卡牌弹层，地图仍可在周边看到。
- “完整”模式继续使用全屏档案页，简述／完整可即时切换。
- 点击简述卡牌外部背景或按 Esc 返回地图。


## v021

- 新增“新包 / 已查看未应用 / 已应用”状态。

## v022

- 可更新简要、基础、详细及地块环境、文明、神话现象和十八经篇事件。
- 对象与地块档案可在一个 Markdown 文件中混合导入。
- 支持缩进多行文本；地块档案采用局部字段更新，不会覆盖未写字段。
- 修复“载入空白模板”实际载入正确案例的问题。
- 修复 app 页面仍显示 v020、data/current.json 仍显示 v020 的版本信息不一致。
- 重新生成 SHA256 校验清单。


## v023 · 高倍率彩色精细地图
- 420%以上自动进入10里彩色精细地图，380%以下返回100里地块卡片。
- 10里小格根据山地、丘陵、林地、平原、水域、湿地、荒漠、冰雪、聚落等数据着色。
- 对象按真实里坐标显示彩色标记；同一精确坐标的多个对象合并为可展开对象组。
- 明确录入路径节点的河流显示连续蓝色与重复流向箭头。
- 面积与作用域按证据等级、透明度和边界样式叠加；未录入路径的数据不自动连线。
- 高倍率空白处双击，可按当前精确坐标新增对象。


## v024 · 地貌与水系叠层
- 高倍率地图中，山地、丘陵等继续作为基础底色。
- 河流、溪流、泉源和水体改为独立蓝色几何叠加，不再覆盖基础地貌。
- 同一精确坐标同时存在山丘与水系时，标记使用山色与蓝色双色分区。
- 明确水体显示半透明蓝色面积；泉源显示径向水源斑；路径未定只显示小型水痕。
- 候选或项目推定路径使用低透明度虚线，明确路径继续显示流向箭头。


## v025
- 高倍率精细地图中，河流与溪流改为线条优先显示。
- 河流对象不再参与10里格底色填充，避免出现一块块蓝色方块。
- 山丘、山地等地貌保留底色，水系以独立蓝色线条叠加。


## v027 · 宽幅地块简述、画笔集合与教学
- “简述”改为宽幅地块身份卡，地图仍保留在卡牌周边。
- 地块内容按山水地貌、草木、鸟兽、金玉矿物、人群神祇、事件遗迹六类排列。
- 分类标题只在有对象时显示实际数量。
- 同一精确坐标的对象不再显示重叠数量徽标，改为“名称 / 名称”完整列出。
- 重叠卡片新增翻转按钮，背面可以切换查看各对象的独立资料。
- 新增地块采集画笔；划过的100里地块自动组成可滚动的简述集合。
- 集合支持继续追加、擦除、清空、地图定位、移除和打开完整博物志。
- 新增首次打开引导与使用教学中心。
- 地块档案和 Markdown 导入新增一句话摘要、关系、可信度、水文、证据链、定位结论等纯地图调研字段。
- 界面不再展示旧版扩展字段；历史本地数据仍保留以避免升级丢失。
