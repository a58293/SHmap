山海经原典地图研究台 v004 / 0.4.4

本包基于 v0.4.3 完整源码更新。

本次更新：
1. 修复“这里有什么”多分类窄列、逐字换行、内容重叠与裁切。
2. 关系线显示关系标签与图例；悬停显示依据；单击关系线跳转到另一端对象。
3. 420%以上显示当前位置、网格坐标、中心坐标与最近地标方向；空旷区域显示位置水印。
4. 默认不显示空白地块；隐藏空白地块时，单击地图空白背景仍可退出聚焦。
5. 地图内右键可退出区域／地块聚焦。
6. 画笔集合页面改为与“这里有什么”相同的分类卡片。
7. 画笔模式单击右键撤回上一次笔画；双击右键清空全部集合。
8. 画笔按钮旁新增“取消画笔状态”，退出并清除画笔线，但保留已采集地块。
9. 保留九段式 Markdown 导入、无回弹缩放、精确对象悬停窗和完成编辑自动同步 GitHub。

覆盖更新：
- 关闭地图程序和正在运行的开发窗口。
- 备份 F:\SHmap\SHmap。
- 将覆盖更新包内文件复制到 F:\SHmap\SHmap。
- 出现同名文件提示时选择“替换目标中的文件”。

运行校验：
cd F:\SHmap\SHmap\desktop-app
npm ci
npm run verify
npm run desktop:dev
