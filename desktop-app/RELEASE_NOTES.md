# v0.6.1 · 关系说明与工具栏可用性修复

- 单击关系线不再立即跳转，而是在关系线上打开说明卡。
- 说明卡展示起点、终点、关系类型、线型、方向、证据与推导说明，并提供跳转按钮。
- 关系线命中范围扩大，细线更容易点击。
- 关系线优先从对象卡片边缘出发，并进行轻量障碍避让。
- 同坐标对象使用分散接口，减少中心标签与线路堆叠。
- 屏外聚合端点显示“方向＋数量”，不再只显示裸数字。
- 顶部关系工具分组，低频操作收纳到“更多”，避免右侧按钮被裁切。
- 工具按钮停留约0.8秒显示用途说明。
- 关系图例可以折叠；退出关系模式入口更明确。
- 地块文字缩放设置上限，地图继续放大时名称不会无限变大。

# SHmap v0.6.0｜关系网络与浏览体验整合版

- 关系网络提供“精简关系／全部关系”双模式。
- 同方向关系先合流再分叉，减少中心放射状拥挤。
- 屏幕外目标按方向聚合，点击数量端点查看目标列表。
- 默认只保留少量关系标签，悬停显示完整关系说明。
- 新增对象聚焦、研究视图／展示视图切换。
- 对象编辑器新增图片选择、WebP压缩、本地程序数据目录存储、来源与版权备注。
- 删除前显示关系、水系、区域及文本提及影响。
- 保留0.5.4关系拖动稳定、区域范围与白块隔离修复。
