# 山海经原典地图研究台 v0.4.3

- 新增 01—09 九段式 Markdown 博物志导入。文件名与客户端既有对象名称匹配后，直接更新该对象博物志。
- 九段式导入不会新建坐标、移动对象或覆盖原有几何定位。
- 第06节支持地貌、水系、草木、鸟兽、金玉矿物、人群神祇、事件与遗迹。
- “原文未载明确”等占位文字自动忽略，不生成空白卡片。
- 分类条目能唯一匹配现有对象时自动绑定；歧义或缺失项保留为文字卡片并标记未绑定。
- 重复导入完全相同的文件不会重复生成修改记录。
- 原有“### 对象：”和“### 地块档案：”Markdown 导入格式继续保留。
- 精确对象预览窗的滚轮区域改为智能路由：列表可滚动时滚动列表，到达边界或无溢出时继续缩放地图。
- 保留 v0.4.2 的博物志分类滚动、精确点详情、无回弹缩放、悬停预览和完成编辑自动同步 GitHub。

本地校验：

```powershell
npm ci --registry=https://registry.npmjs.org/
npm run verify
npm run build
npm run desktop:dev
```

正式发布标签：`desktop-v0.4.3`。
