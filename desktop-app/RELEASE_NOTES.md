# 山海经原典地图研究台 v0.5.3

v0.5.3以v0.5.2世界／区域／地点层级版为基线，修复区域概览的白色碎块、伪边界、标签重建和跨区水系范围膨胀问题，并重新区分水系方向证据与非水系线对象。

## 区域概览

- 区域概览不再绘制100里格逐格遮罩。
- 拖动和缩放时保留搜索与聚焦反馈。
- 区域标签复用DOM节点，不再每帧整体销毁重建。
- 退出区域概览后恢复普通操作提示。
- 无真实边界的区域不再显示实心椭圆。

## 层级与范围

- 区域名称使用严格分隔规则，减少碎片化区域。
- 模块、路径、异文、候选和项目描述不再默认作为主要地理区域。
- 区域范围按成员对象的点位或自身面积计算；跨区河流整条路径不再扩张每个区域。
- 世界大区仍是项目坐标导航层，不作为《山海经》原文明载边界。

## 线型对象与水系

- 边界、山系、路线、关系线采用不同线型和颜色。
- 只有明确标为有向的非水系线才显示箭头。
- 水系方向拆分为原文方向、拓扑方向、项目排版与未定。
- G3项目示意路径不显示确定流向箭头。

正式版本：`0.5.3`。
