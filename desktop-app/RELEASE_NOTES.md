# 山海经原典地图研究台 v0.4.5

本版从关系筛选缺失问题开始，集中更新关系分类、博物志标签检索和对象索引排序。

## 关系分类重构

- 每条关系根据关系类型和两端对象类型进行多标签归类，可同时属于多个主题。
- 主题包括：地理空间、水系流向、文明人群、神话生物、事件遗迹、异名同体、其他关系。
- “其他关系”作为兜底，保证全部关系中的内容不会在主题视图中无故消失。
- 明确原文、空间推定和同篇关联改为独立的第二层证据筛选。
- 每个主题按钮显示当前关系数量；切换主题或证据层级后立即重绘。
- 关系线悬停卡片显示关系名称、目标对象、主题和证据层级，点击跳转功能保持不变。

## 可点击博物志标签

- 本地标签、父级区域、相邻地块、关联水域和关联生灵均可点击。
- 点击后打开同标签索引，分别显示相关地块和对象。
- 点击结果可直接定位；“只看地图”会将标签写入地图检索。
- 右键、Esc、遮罩或关闭按钮均可退出标签索引。

## 对象索引排序

- 增加默认顺序、最近更新、最早更新、本轮修改优先、名称、经篇、类型和距离当前位置。
- 支持升序／降序切换和“仅本轮”筛选。
- 排序状态保存到工作区；排序后当前选中对象保持高亮并自动滚动到可见位置。
- 对象编辑、Markdown 导入和批量更新会写入真实更新时间。旧对象没有更新时间时不会生成虚构日期。

本版继续保留 v0.4.4 的关系线置顶与点击跳转、高倍定位、空白地块默认关闭、画笔右键撤回／清空和博物志卡片滚动。
