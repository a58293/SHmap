# 山海经原典地图研究台 v0.5.1

v0.5.1 以“特大字号与非阻挡通知修复版”0.5.0为唯一开发基线，完成 v125 母表同步和线型水系重构。

## 数据升级

- 地图对象：395 → 617。
- 数据版本：v075-r0001 → v125-r0001。
- 原文总库：2432 → 4120条记录。
- 从坐标棋盘图读取118个河流箭头格，生成79段可交互线型水系。
- 同步保存G1—G4证据等级、原文硬距离和地图几何距离。
- 旧工作区升级时保留本地新增对象与本轮已编辑对象。

## 线型水系

- 普通河流不再用整块蓝色地块表示。
- 海、泽、湖、渊、池等面积水体继续使用浅蓝面积。
- 河流显示源点、路径、流向箭头、汇入端和证据线型。
- G1、G2使用实线；G3使用点划线；G4使用虚线。
- 鼠标悬停可查看河名、经篇、区域、流向和证据等级。
- 单击河流可高亮整段、缩放到完整路径，并在右侧显示水系档案。
- 水系视图提高河流可见度，其他视图降低密度，避免遮挡对象。

## 继承的界面修复

- “特大”字号继续明显放大正文、按钮、关系标签、地图标注、悬停卡片和详情标题。
- 特大字号同步增加顶部栏和工具栏高度，避免文字裁切。
- 操作提示继续位于底部中央，并保持点击穿透。
- 普通提示约2.4秒、错误提示约4.8秒；同时最多保留3条。

## 母表导入命令

```powershell
npm run import:master -- "新版母表.xlsx"
```

导入脚本会生成对象数据、水系路径和 `water-path-validation.json` 校验报告。
