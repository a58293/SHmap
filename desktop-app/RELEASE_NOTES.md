# 山海经原典地图研究台 v0.4.4

本版集中修复博物志排版、关系导航、高倍缩放定位与画笔采集交互。

## 博物志排版

- “这里有什么”不再把五至六个分类强行挤在同一行；桌面端最多三列，窗口变窄时自动切换为两列或单列。
- 对象卡片名称、类型、核心特征、功效与证据保持正常横排，避免逐字换行、重叠和裁切。
- 每个分类仍保留独立纵向滚动；无图片对象继续显示“待补图”。
- 九段式 Markdown 导入形成的未绑定博物志条目继续显示，不会创建或移动地图坐标。

## 关系高亮导航

- 关系高亮线增加关系名称标签和固定图例：实线表示明确／直接关系，虚线表示同地块、同区域或同篇等关联。
- 悬停关系线显示当前对象、目标对象、关系类型和识别依据。
- 关系线实际点击范围扩展到约14像素；单击后平滑跳转到线另一端，自动选中目标并更新右侧详情。
- 按下后发生明显移动仍按地图拖动处理，不会误触关系跳转。

## 高倍缩放定位

- 420%以上显示当前位置面板，包括当前区域、100里网格、中心坐标和缩放比例。
- 自动列出最近地标的方向、距离和类型，单击即可平滑跳转。
- 空旷画面中央显示低透明度区域／网格水印；位置面板可一键返回地图总览。

## 空白地块与聚焦退出

- v0.4.4 首次启动时默认关闭“空白地块”，减少大面积空卡片干扰；用户之后的选择会正常保存。
- 即使空白地块不显示，单击地图空白背景仍会退出区域／地块聚焦。
- 在地图内单击右键也可退出当前聚焦状态。

## 画笔采集

- 画笔集合页面改为与地块博物志一致的“这里有什么”分类陈列。
- 单击右键撤回上一次笔画，同时恢复该笔画造成的地块选择变化。
- 双击右键清空全部集合和画笔轨迹。
- “画笔采集”旁新增“取消画笔状态”，用于退出画笔并清除轨迹；已经采集的地块仍保留。
- 完成一笔后保持在地图画笔模式中，不再自动弹出集合页；通过“地块集合”按钮主动查看结果。

本版继续保留 v0.4.3 的 01—09 九段式 Markdown 博物志导入、智能滚轮、无回弹缩放和自动同步 GitHub。
