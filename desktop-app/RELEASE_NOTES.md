# 山海经原典地图研究台 v0.5.2

v0.5.2 在 v0.5.1 线型水系版基础上，增加世界／区域／地点层级。当前目标是让内部成员快速理解全图结构，不实装游戏任务、区域等级、敌人、资源或气候生态内容。

## 世界与区域层级

- 建立固定世界根节点“山海经世界”。
- 根据现有区域文字与坐标方位生成5个世界大区、266个地图区域。
- 世界大区和地图区域均支持父子关系、中心、范围、成员对象和涉及经篇统计。
- 区域环境字段 `environmentProfile` 与边界字段 `boundary` 已预留为空。
- 区域分类不替代原有经篇分类。

## 对象归属

- 每个对象新增 `primaryRegionId`，用于默认区域、检索和层级路径。
- 每个对象新增 `regionIds`，允许河流、山系和跨区对象关联多个区域。
- 未分区对象会进入待归区结构，不会丢失或隐藏。
- 母表导入时会重新生成区域归属、世界层级和区域统计。

## 界面与导航

- 左侧索引新增“对象／世界·区域”切换。
- 世界大区支持折叠，地图区域支持关键词检索。
- 点击世界、大区或区域会自动缩放到对应范围。
- 区域聚焦会保留本区域对象并降低其他区域干扰。
- 右侧新增世界详情与区域详情，显示上级区域、成员数量、涉及经篇和成员地点。
- 地点详情增加“世界 > 大区 > 区域 > 地块 > 地点”可点击路径。
- 低倍缩放优先显示区域名称，高倍缩放恢复具体地点。
