# 山海经原典地图研究台 v0.4.2（整合版）

- 简述博物志的分类栏增加独立纵向滚动区，单个分类对象过多时不再把整页无限拉长。
- 420%以上精细地图中的对象点支持精确交互：悬停时为同一坐标下的每个对象分别显示预览小窗。
- 单击多对象点可固定或取消固定预览组；单击具体小窗打开对应对象的精确点博物志，双击可直接编辑对象。
- 精确预览靠近地图右侧或底部时自动换方向展开；同坐标对象较多时，预览区内部出现滚动条。
- 修复滚轮放大、缩小停止后的视觉回弹。地图画布、地块和精确对象统一使用相机状态逐帧重绘。
- 滚轮改为连续缩放曲线，缩放中心固定在鼠标指针位置；程序内定位动画使用相同的无回弹渲染。
- 取消进入或退出区域概览、10里精细地图时的淡入跳动。
- “完成本轮编辑”会先保存桌面数据库，再生成 `.shjpatch`、写入 `submissions/pending` 并使用本机 Git 登录自动推送 GitHub。
- 自动上传失败时保留本轮更改，并提供重试或仅导出本地文件。

```powershell
npm ci --registry=https://registry.npmjs.org/
npm run verify
npm run build
npm run desktop:dev
```

本整合版继续使用版本号 `0.4.2`，不需要再安装单独的 `0.4.3`。
