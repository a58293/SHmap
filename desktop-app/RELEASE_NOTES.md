# 山海经原典地图研究台 v0.5.0

v0.5.0 将原有地图功能整合为更明确的研究工作台。

## 主要更新

- 新增统一模式状态条和互斥模式管理。
- 新增左右侧栏折叠与专注地图。
- 新增地图浏览历史、前进后退与研究书签。
- 新增 `Ctrl+K` 全局检索。
- 新增本轮操作历史、撤销与重做。
- 关系系统增加直接、空间推定、同篇关联三层显示。
- 对象详情增加证据等级、来源类型和更新时间摘要。
- 重组地图工具和系统操作层级。
- 关系绘制在拖动期间自动降载。

## 兼容性

- 保留 v0.4.5 的 395 个对象。
- 保留数据版本 v075-r0001。
- 保留九段式 Markdown 导入、关系线导航、标签检索、对象排序、画笔采集和桌面更新功能。
