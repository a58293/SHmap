# 山海经原典地图研究台 · 桌面版 v0.5.1

本版本基于 v0.4.5，将地图浏览、资料编辑和关系研究整合为统一工作台。

核心功能包括：

- 无限 100 里棋盘与 10 里精细视图；
- 地块博物志、对象博物志和九段式 Markdown 导入；
- 关系主题、证据等级和直接／空间／同篇三层显示；
- 画笔采集、测量、对比、标签检索和对象排序；
- 统一模式状态条、专注地图、全局检索、浏览历史和研究书签；
- 本轮操作历史、撤销与重做；
- SQLite 桌面存储、备份、GitHub 数据发布和签名更新器。

## 开发运行

```powershell
npm ci
npm run verify
npm run desktop:dev
```

## 构建

```powershell
npm run build
npm run desktop:build
```

正式发布标签：`desktop-v0.5.1`。
