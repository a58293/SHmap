# 山海经原典地图研究台 · 桌面版 v0.5.2

本版本在 v0.5.1 的617个对象与线型水系基础上，实装“世界／区域／地点”层级导航。它用于内部查看整个《山海经》世界的结构，不加入任务、敌人、等级等游戏玩法。

核心功能包括：

- 无限100里棋盘与10里精细视图；
- 617个地点对象、4120条原文记录与79段线型水系；
- 固定世界根节点、5个世界大区和266个地图区域；
- 左侧“对象／世界·区域”双索引；
- 区域树、区域检索、点击定位、区域聚焦与世界总览；
- 地点的主要区域与多区域关联；
- 世界／大区／区域／地块／地点层级路径；
- 低倍区域总览与高倍地点显示；
- 地块博物志、对象博物志和九段式 Markdown 导入；
- 关系研究、画笔采集、测量、对比、标签检索和对象排序；
- SQLite桌面存储、备份、GitHub数据发布和签名更新器。

气候与生态字段已预留为空，本版本不生成相关内容。地图区域属于内部空间整理层，与《海内经》《大荒东经》等经篇分类相互独立。

## 开发运行

```powershell
npm ci
npm run verify
npm run desktop:dev
```

## 构建

```powershell
npm run build
npm run desktop:build
```

## 更新母表

```powershell
npm run import:master -- "新版母表.xlsx"
```

导入脚本会重新生成对象数据、线型水系、世界区域层级和校验报告。

正式发布标签：`desktop-v0.5.2`。
