# CHANGELOG

## Desktop v004.5

- 关系视图改为多标签归类：同一关系可同时进入地理空间、水系流向、文明人群、神话生物、事件遗迹和异名同体。
- 增加“其他关系”兜底，确保“全部关系”中的每条关系至少进入一个主题分类，不再只在全部视图出现。
- 证据性质改为第二层筛选，可独立查看明确原文、空间推定和同篇关联；主题按钮实时显示关系数量。
- 关系线悬停说明增加主题分类和证据层级，原有点击跳转、关系标签和线层置顶保持不变。
- 地块博物志中的本地标签、父级区域、相邻地块、关联水域和关联生灵改为可点击标签。
- 点击标签后打开同标签索引，分别列出相关地块与对象；结果可直接定位地图，也可切换为地图检索。
- 左侧对象索引增加默认、最近更新、最早更新、本轮修改、名称、经篇、类型和距离当前位置排序。
- 排序支持升降序与“仅本轮”，选择会随工作区保存；对象编辑和 Markdown 导入自动记录真实更新时间。

## Desktop v004.4

- 修复“这里有什么”在多分类时被强行压缩的问题：桌面端最多三列，窄窗口自动切换为两列或单列。
- 对象卡片保持正常横向中文排版，名称、类型、核心特征与证据不再逐字竖排、重叠或裁切。
- 保留每个分类栏独立滚动；空图片继续显示“待补图”，九段式未绑定文字条目继续保留。
- 关系高亮线增加在线关系标签与固定图例，区分明确／直接关系和同地块／同区域／同篇关联。
- 悬停关系线显示起点、目标、关系类型与依据说明，并扩大到约14像素的实际点击范围。
- 单击任意关系线后平滑跳转到另一端对象，自动选中目标并刷新右侧详情；拖动地图不会误触跳转。
- 420%以上增加高倍缩放定位面板：显示当前区域、100里网格坐标、中心里坐标及最近地标方向与距离。
- 精细地图空旷区域增加低干扰位置水印；点击最近地标按钮可直接定位，点击“地图总览”可回到全图范围。
- “空白地块”改为首次进入 v0.4.4 时默认关闭；之后用户手动切换的状态会正常保存。
- 空白地块隐藏后，单击无对象的地图背景仍可退出区域／地块聚焦；右键也可立即退出聚焦。
- 画笔采集的汇总页面改用与地块博物志一致的“这里有什么”分类卡片。
- 画笔模式中单击右键撤回上一次笔画及其选区变化，双击右键清空全部采集地块和画笔线。
- “画笔采集”旁新增“取消画笔状态”：退出画笔模式并清除画笔线，但保留已经采集的地块集合。
- 保留 v0.4.3 的九段式 Markdown 博物志导入与既有 v0.4.2 交互修复。

## Desktop v004.3

- 新增 01—09 九段式 Markdown 博物志导入，按文件名匹配客户端既有对象。
- 九段式导入只更新对象博物志，不新建坐标、不移动对象，也不覆盖原有地图定位。
- 第06节支持地貌、水系、草木、鸟兽、金玉矿物、人群神祇、事件与遗迹七类条目。
- “原文未载明确”等占位内容不会生成空卡片；无法唯一绑定的分类条目会保留为博物志文字卡片并标记未绑定。
- 重复导入内容完全相同的文件不会重复生成修改记录。
- 保留 v0.4.2 的博物志分类滚动、精确点详情、无回弹连续缩放、对象悬停预览和完成编辑自动同步 GitHub。
- 修复精确对象预览窗的滚轮区域：列表可滚动时滚动列表，到达边界后继续缩放地图。

## Desktop v004.2（整合版）

- 简述博物志的分类栏增加独立纵向滚动区，单个分类对象过多时不再把整页无限拉长。
- 420%以上精细地图中的精确点支持打开博物志；同坐标多对象先选择具体对象，再打开精确点档案。
- 精确点博物志仅展示同一精确坐标的对象，并保留简述、完整、定位和复制摘要功能。
- “完成本轮编辑”会先保存桌面数据库，再生成 `.shjpatch`、提交到 `submissions/pending` 并推送 GitHub。
- 自动上传不内置密钥，复用本机 GitHub Desktop / Git 登录；网络失败时保留本轮更改并提供重试与本地导出。
- 上传前检查 main 分支与远端同步状态，避免顺带推送未确认的代码提交。
- 地图滚轮缩放与程序内定位动画改为逐帧真实重绘，移除临时缩放图层回基准导致的视觉回弹。
- 缩放手感改为按滚轮增量连续计算，鼠标指针下的地图位置保持稳定。
- 进入／退出区域概览和10里精细地图时取消淡入动画，模式切换不再出现二次缩放错觉。
- 精细地图悬停对象点时，为同一精确坐标下的每个对象分别弹出独立预览小窗。
- 单击多对象点可固定／取消固定预览组；单击任一小窗打开对应精确点博物志，双击可编辑该对象。
- 预览组靠近窗口边缘时自动向左或向上展开，并在对象较多时内部滚动。

## Desktop v004.1

- 增加仓库直连、jsDelivr CDN 与 GitHub Releases 三条更新检查线路。
- 每条检查线路自动重试，连接重置或超时后自动切换下一线路。
- 更新包下载增加最多3次自动重连，避免短暂断网直接导致安装失败。
- 发布成功后自动同步签名 `latest.json` 到仓库稳定更新源。
- 更新窗口显示当前使用线路、重试状态与完整失败原因。

## Desktop v004

- 仅重做地块博物志“简述”模式，完整模式与原文证据区保持原有逻辑。
- 移除简述页自动摘要和固定六类概览，改为只显示有对象的分类。
- 每个对象卡片增加独立图片区、名称、关系、核心特征、功效与证据摘要。
- 没有图片或图片加载失败时自动显示统一“待补图”占位，不出现破图标记。
- 对象编辑与 Markdown 导入增加可选图片字段，为后续图片素材导入预留绑定接口。
- Markdown 工作台增加单文件与批量文件选择；批量导入显示文件清单与条目来源。
- 批量选择限制为最多100个文件、单文件5MB、总量25MB，避免一次载入过量文本。
