# 桌面版 v001 功能与验收说明

## 核心目标

第一阶段先解决浏览器缓存、数据保存、备份恢复和安装运行问题。地图视觉与现有网页保持一致，避免同时重写全部交互。

## 数据安全

- SQLite 使用 WAL 模式。
- 每次界面保存都会更新当前工作区。
- 内容发生变化且距离上次自动备份达到60分钟时建立自动备份。
- 恢复旧备份之前自动保存“恢复前备份”。
- 每份数据库备份同时生成可读的 `.shjbackup.json` 文件。
- 自动备份最多保留最近48份，避免磁盘无限增长；手动备份和恢复前备份不自动删除。
- “检查数据库”运行 SQLite `PRAGMA quick_check`。

## 首次启动

数据库为空时按以下顺序初始化：

1. 可用的旧兼容缓存；
2. 内置 v075 数据种子。

## 验收项目

- 程序可离线启动。
- 首次启动显示395个对象。
- 编辑后状态显示“已写入桌面数据库”。
- 手动备份可创建并列出。
- 恢复备份后程序重新载入对应工作区。
- 关闭并重启后数据仍存在。
- 数据库检查结果为 `ok`。
