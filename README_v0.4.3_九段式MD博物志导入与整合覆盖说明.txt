山海经原典地图研究台 v004 / 0.4.3
九段式 Markdown 博物志导入与整合覆盖说明

本版本以当前完整 v0.4.2 为基线，保留：
1. 博物志分类内部滚动。
2. 精细地图精确对象悬停预览和精确点博物志。
3. 无回弹连续缩放。
4. 完成本轮编辑后自动生成补丁并同步 GitHub。
5. 精确对象预览窗智能滚轮区域。

新增：
1. 支持 01—09 九段式 Markdown。
2. 文件名匹配客户端既有对象，导入只更新对象博物志。
3. 第06节支持地貌、水系、草木、鸟兽、金玉矿物、人群神祇、事件与遗迹。
4. “原文未载明确”不会生成空卡片。
5. 能唯一匹配的分类条目自动绑定地图对象；无法唯一匹配的内容保留为文字卡片。
6. 相同文件重复导入不会重复生成修改记录。

覆盖更新：
把本压缩包解压到 F:\SHmap\SHmap，选择“替换目标中的文件”。
然后进入 F:\SHmap\SHmap\desktop-app，在 PowerShell 中运行：

npm run verify
npm run desktop:dev

校验应显示：程序 v004 / 0.4.3。
确认功能正常后，再提交 GitHub 并发布 desktop-v0.4.3。

不要删除或覆盖本机已有的 SQLite 工作数据、签名私钥或 GitHub 登录信息。源码覆盖不会删除已安装客户端中的数据库。
